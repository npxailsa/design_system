# Label Component

The `Label` component is a versatile text display component that supports icons, alignment, and dropdown split functionality. It uses the **F37 Ginger Pro** font family as its primary typeface, ensuring a modern and clean aesthetic that matches our design system.

## Typography and Scaling

The component is built using Figma-compliant design tokens. Each size variant scales the text, the icons, and the spacing (gap) between elements proportionally:

- **large**: 18px text, 18px icon, 8px gap.
- **default**: 16px text, 16px icon, 6px gap.
- **small**: 14px text, 14px icon, 4px gap.
- **x-small**: 12px text, 12px icon, 4px gap.
- **2x-small**: 10px text, 10px icon, 2px gap.

## Usage

```tsx
import { Label } from './components/Label';
import MailIcon from '@mui/icons-material/Mail';

<Label size="large" weight="bold" leadingIcon={MailIcon}>
  Email Support
</Label>
```

## Component API

The `Label` component is highly configurable with several props to control its appearance and behavior:

- **children**: The text to display. Defaults to 'Label'.
- **size**: Controls font size, line height, gap, and icon sizes. Supports 'large', 'default', 'small', 'x-small', '2x-small'.
- **weight**: Controls the font weight. Supports 'bold', 'medium', 'light', 'thin'.
- **align**: Controls the text and content alignment within the container. Supports 'left', 'centre', 'right'.
- **dropdown**: If true, displays a vertical divider and a chevron-down icon.
- **leadingIcon**: A Material UI icon to display before the text. Defaults to `PersonIcon`.
- **showLeadingIcon**: If true, displays the leading icon.
- **showTrailingIcon**: If true, displays an `ArrowForwardIcon` after the text.
- **showSplit**: If true, displays the split divider and chevron (only when `dropdown` is true).
