# Design Tokens: Colors

Our color palette is structured into three layers: **Global** (primitives), **Brand** (contextual aliases), and **Component** (semantic). Always prefer using component or brand tokens over global ones.

## Usage

When using colors in your components, always refer to the design tokens defined in `tokens.css`.

```css
.my-component {
  background-color: var(--global-color-base-white);
  color: var(--brand-text-main);
  border: var(--global-spacing-stroke-1px) solid var(--brand-border-main);
}
```
