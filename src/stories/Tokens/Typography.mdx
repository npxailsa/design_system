# Design Tokens: Typography

Our typography system is structured into global primitives, brand aliases, and semantic component tokens.

## Global Typefaces

We use two primary font scales for our projects: **F37 Ginger Pro** and **Calibri**.

- **Global Primary Font**: `var(--global-font-primary)` (F37 Ginger Pro)
- **Global Secondary Font**: `var(--global-font-secondary)` (Calibri)

## Brand Aliases

These tokens represent the active brand theme mappings for the NPX Design System.

- **Brand Primary Font**: `var(--brand-font-primary)`
- **Brand Secondary Font**: `var(--brand-font-secondary)`

## Font Weights

Always use the standard font weight tokens for consistency:

- **Thin**: 100 (`var(--global-type-weight-thin)`)
- **Light**: 300 (`var(--global-type-weight-light)`)
- **Default/Regular**: 400 (`var(--global-type-weight-default)`)
- **Medium**: 500 (`var(--global-type-weight-medium)`)
- **Semibold**: 600 (`var(--global-type-weight-semibold)`)
- **Bold**: 700 (`var(--global-type-weight-bold)`)
- **Extra Bold**: 800 (`var(--global-type-weight-extra-bold)`)

## Usage

When using typography in your components, always refer to the design tokens defined in `tokens.css`.

```css
.my-title {
  font-family: var(--brand-font-primary);
  font-size: var(--global-type-size-primary-heading-1);
  font-weight: var(--global-type-weight-bold);
  line-height: var(--global-type-line-height-heading-1);
}

.my-text {
  font-family: var(--brand-font-secondary);
  font-size: var(--global-type-size-secondary-body);
  font-weight: var(--global-type-weight-default);
  line-height: var(--global-type-line-height-body);
}
```
